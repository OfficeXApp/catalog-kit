## About

This repository contains the skill documentation (SKILL.md) for the **catalog-funnel** app on OfficeX.

## What is a SKILL.md?

A SKILL.md is a structured documentation file that AI agents (like Claude, ChatGPT, etc.) can consume to understand how to interact with an app's API.
